So here’s an encoding method that can render an encrypted message to be only encoded:

  1. Encrypt message (any valid encryption) with key “A”, _where “A” is a valid encoding method used in step 2

  2. Encode message with encoding method A

There you go, you can now render that great security by encryption completely useless :)
